# 🚀 My Project Starters

Колекція готових конфігурацій для швидкого старту нових проєктів.
Кожна конфігурація зберігається в окремій гілці.

## 🚀 Як користуватися

1. **Створіть порожню папку** або склонуйте новий репозиторій:

    ```bash
    mkdir my-new-app
    cd my-new-app
    git init
    ```

2. Розгорніть потрібний шаблон (замініть #branch на назву гілки):

    ```bash
    npx degit https://github.com/atoptun/starting-templates#branch . --force
    ```

3. Виконайте інструкції з readme для корегування
